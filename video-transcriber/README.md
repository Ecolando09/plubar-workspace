# Video Transcriber Android App

Simple Android app to transcribe audio from video files using ElevenLabs API.

## Features
- Select local video file
- Extract audio and transcribe using ElevenLabs Scribe v2
- Save transcription as .txt file

## Setup

1. Open in Android Studio
2. Build and run on device/emulator
3. API key is pre-configured in BuildConfig.java

## Requirements
- Android SDK 24+ (Android 7.0+)
- Internet permission for API calls
